# Exercise 01: Pod basics

**Type:** Build  
**Difficulty:** Beginner

## Goal

Create a simple pod running nginx and verify it's working.

## Tasks

1. Create a pod named `my-first-pod`
2. Use the `nginx:latest` image
3. Verify the pod is running

## Success criteria

- Pod `my-first-pod` exists in the default namespace
- Pod status is `Running`
- Container is using nginx image

## Hints

Use `kbx hint 01` if you're stuck!
